# Kei-Tracker Project

This project aims to fix bugs, add new features, and update documentation for the Kei-Tracker application. Key features include real-time updates and database management, targeting end-users with a modern and minimalist UI aesthetic.

## Purpose

- Fix existing bugs
- Add new features (Real-time Updates, Database Management)
- Update documentation

## Key Features

- **Real-time Updates:** Live data updates for ship positions.
- **Database Management:** Efficient storage, retrieval, and management of ship data.

## Target Audience

- End Users

## UI Aesthetic

- Modern & Minimalist
